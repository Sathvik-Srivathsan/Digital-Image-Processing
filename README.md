# OpenCV Image Processing Notebook

A personal OpenCV notebook with a clean set of image processing experiments.  
It covers transforms, histogram work, filtering, noise, restoration, and frequency domain methods in one place.

## What is inside

- Image transforms
- Gray level transformations
- Histogram processing
- Spatial filtering
- Smoothing filters
- Sharpening filters
- Noise generation and analysis
- Restoration with spatial filters
- Periodic noise reduction in the frequency domain
- Degradation function estimation
- Inverse filtering

## Tech stack

- Python
- OpenCV
- NumPy
- Matplotlib
- SciPy
- PyWavelets

## How to use

1. Open the notebook in Jupyter or VS Code.
2. Install the required packages.
3. Run the cells in order.

## Project structure

- `DIP_Assignment.ipynb` - main notebook with all experiments
- `README.md` - project overview
- `human.jpg` - input image

## Input Image

The notebook uses an input image named human.jpg.

Before running the notebook, place your image in the project root directory and rename it to:

human.jpg

Or change the image file name to the name of the uploaded image file.

Update the path if necessary.
